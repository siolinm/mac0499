Proposta de trabalho

O tema/assunto a ser trabalhado é o estudo de problemas cinéticos e como eles são resolvidos eficientemente através do uso de estruturas de dados cinéticas.

Problemas cinéticos são problemas em que desejamos manter atributos geométricos de um conjunto de objetos que estão em movimento contínuo. Por exemplo, dado um conjunto de pontos em movimento, rastrear um par de pontos que possui distância mínima.

A trajetória desses objetos pode ser um polinômio de grau limitado, mas para este trabalho estaremos nos limitando ao estudo de trajetórias lineares.

Objetivos do trabalho

Estudar problemas cinéticos e as estruturas de dados utilizadas para resolver estes problemas. Fazer implementações das estruturas e, nos problemas em que é pertinente, uma animação para visualização dos acontecimentos.

Dentro dos estudos das estruturas cinéticas, apresentar medidas da performance delas:

  • Responsividade
  • Localidade
  • Compacidade
  • Eficiência

Problemas a serem estudados:

  • Problema da Ordenação Cinética:
    • Lista Ordenada Cinética
    • Árvore binária de busca balanceada cinética
  • Problema do Máximo Cinético:
    • Heap Cinético
    • Torneio Cinético
  • Problema do Par Mais Próximo Cinético:
    • Algoritmo Estático (algoritmo que resolve a versão estática do problema)
    • Algoritmo Cinético
  • Triangulação de Delaunay:
    • Versão Estática
    • Versão Cinética
